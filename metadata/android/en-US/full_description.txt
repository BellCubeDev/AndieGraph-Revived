AndieGraph Revived is a faithful calculator emulator for TI-85, 82, 83, 83+ and 86 calculators. It uses user-provided ROMs to emulate the calculator's functionality perfectly. The emulator is designed to be as accurate as possible, providing a seamless experience for users who want to use their favorite calculator on their Android device.

NOTE: this emulator requires user-provided ROMs to function. Please ensure you have the appropriate ROMs for your calculator model before using the emulator. Additionally, these ROMs must contain "TI" and the calculator model number in their filename to be recognized by the emulator; for example, a TI-85 ROM should be named something like "TI-85.rom". The emulator will not function without the correct ROMs. There are many great guides for dumping ROMs from your calculator online, such as the master guide from ticalc.org: https://www.ticalc.org/programming/emulators/romdump.html

The app's meta-controls (settings, importing ROMs, etc.) are revealed by swiping from any edge. These mirror modern Android gestures (back gesture, app switcher, notifications), but will work alongside most of them. The available meta-controls are:
- Importing different ROMs for different calculator models (only one ROM imported at a time currently)
- Haptic feedback enable/disable
- "Zoom" feature which removes the calculator bezel and allows for a larger display area. Not recommended since the introduction of edge-to-edge in Android, but still available for those who prefer it.
- Wake lock enable/disable to prevent the device from sleeping while using the emulator

This is a fork of the original AndieGraph repository. With its last update on the Play Store in 2015, the original author no longer maintains the app and it no longer works on modern Android devices. At the request of one of my coworkers, I gave the app minimal updates to make it compatible with modern Android versions and devices.

If you find any bugs or issues with the emulator, please report them on the GitHub repository. Contributions and pull requests are welcome to help improve the emulator and keep it up to date with the latest Android versions. As for me? My work is largely done.
